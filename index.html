About LazyRent

LazyRent is a local rental management application designed to help landlords manage property and rental information efficiently.

Data & Privacy

All data is stored locally on the user's device. Optional synchronization is performed using Apple's private iCloud storage.

The app does not operate any backend servers and does not collect, transmit, or share any personal data.

Membership (Testing Only)

LazyRent includes membership features for testing purposes only.

Test membership access can be activated via invitation codes. Invitation codes are used solely for feature testing and access control.

No payment, purchase, subscription, or external transaction mechanisms are provided within the app.

关于懒虫收租

懒虫收租是一款本地运行的租务管理应用， 用于帮助房东高效管理房源与租金信息。

数据与隐私

所有数据均存储在用户设备本地。 可选择使用 Apple 提供的 iCloud 私有空间进行同步。

应用不设任何服务器， 不收集、不上传、不共享任何用户个人数据。

会员说明（仅用于测试）

测试会员权限可通过邀请码激活， 邀请码仅用于功能测试与访问控制。

应用内不提供任何付费、购买、订阅或外部交易入口。
